Este paquete de idioma modifica los nombres de los componentes para facilitar la detección y el seguimiento de las estadísticas.

También agrega una serie de mejoras visuales.

* Los contratos que otorgan planos tienen la lista de posibles recompensas añadida al final de la descripción.
* Prefijos de advertencia para sustancias ilegales. **[!]**
* "Hephaestanite" abreviado a "Heph".
* "Instability:" / "Resistance:" (Inestabilidad / Resistencia) abreviados a "Instab.:" / "Resist.:" debido a problemas de superposición de texto.


⚠ Falta por añadir algunas listas de planos en las misiones.

## ⬇️ Descarga e instalación
La descarga esta disponible en <ins>**Ingles**</ins> y en <ins>**Español**</ins>.

- Download the latest version here: [**INGLES**](https://github.com/FranLopezZ/ALTF4LangPack/releases/latest/download/AltF4LangPack-ENG.zip)
- Descarga la última versión aquí: [**ESPAÑOL**](https://github.com/FranLopezZ/ALTF4LangPack/releases/latest/download/AltF4LangPack-ESP.zip)

### 💻 Cómo instalar:

1. Extrae el archivo ZIP.
2. Si ya tienes un archivo `user.cfg`, **no lo sobrescribas**. 
    - En su lugar, abre tu `user.cfg` existente y añade la siguiente línea al final:
    - Para la versión en <ins>**ingles**</ins>. `g_language = english`
    - Para la versión en <ins>**__español__**</ins>. `g_language = spanish_(spain)`
    - Guarda el archivo.
3. Copia la carpeta `data` y el archivo (`user.cfg`) en la raíz de la carpeta LIVE de tu juego.
4. Inicia el juego.

## 📋 LEYENDA COMPONENTES 📋

**Formato** : `[NOMBRE COMPONENTE] [CLASE][TAMAÑO][CALIDAD]`

**Abreviaturas de clases:**
- **M** = **Militar**
- **I** = **Industrial**
- **C** = **Civil**
- **R** = **Racing** (La competición ha sido renombrada para evitar conflictos con Civil.)
- **S** = **Sigilo**

**Ejemplos:**
- `Glacier` → `Glacier M1A` (Militar, Size 1, Quality A)
- `Endurance` → `Endurance I1B` (Industrial, Size 1, Quality B)
- `TS-2` → `Wanderer C3C` (Civil, Size 3, Quality C)
- `Targa` → `Targa R1C` (Competicion, Size 1, Quality C)
- `Mirage` → `Mirage S1A` (Sigilo, Size 1, Quality A)
